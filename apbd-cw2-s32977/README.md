W projekcie użyłem klas abstrakcyjnych tak aby trzymać w nich wspólne cechy dla 
wszystkich użytkowników oraz wszystkich rodzajów sprzętu.

Stworzyłem dwie główne klasy czyli DataBase i Service. Postanowiłem oddzielić ich działanie
tak aby DataBase zajmował się przechowywaniem danych i dodawaniem nowych danych do bazy, a 
Service samą obsługą logiki.

Kara za opóźnienie jest liczona w przez podwojenie ceny dzień za każdy dzień spoźnienia.
